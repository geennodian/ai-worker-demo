# AI派遣社員デモ — ニュース収集 → スプレッドシート転記

## 概要

https://ab-hd.co.jp/ のニュース一覧から最新ニュースを収集し、Googleスプレッドシートに転記する。
デモ用途のため、**すべてのブラウザ操作を人間が操作しているように見せる**。

## 環境変数

| 変数名 | 内容 |
|--------|------|
| `DEMO_SHEET_URL` | 転記先 Google スプレッドシートの URL |

## スクリプトパスの解決

```bash
_S=$(python3 -c "
import pathlib
hits = sorted(pathlib.Path.home().glob('.claude/skills/*/scripts/format_news.py'))
print(str(hits[-1].parent) if hits else '')
")
[ -z "$_S" ] && echo "スクリプトが見つかりません。スキルを再インストールしてください。" && exit 1
```

---

## 重要: 人間らしい操作ルール

**このスキルの全てのブラウザ操作で以下を厳守する。**

### クリック
1. クリック対象の要素にまず**ホバー**する（座標を指定してカーソルを移動）
2. **0.3〜0.5秒待機**する
3. その後**クリック**する

### タイピング
1. テキストを**3〜5文字ずつ**に分割する
2. 各チャンクを入力した後、**0.2〜0.4秒待機**する
3. 日本語の場合は文節単位で区切ると自然に見える

### スクロール
- 一度に大きくスクロールしない
- **scroll_amount は 2〜3** で小刻みにスクロールする
- スクロール間に **0.5秒程度の間** を置く

### ページ遷移
- ナビゲート後、**1〜2秒待機**してからスクリーンショットを撮る
- いきなり操作を始めず、まずページ全体を「見渡す」ようにスクリーンショットを撮る

### セル移動（スプレッドシート）
- セルへの入力後は **Tab キー** で次のセルに移動する
- 行の末尾では **Enter キー** で次の行の先頭に移動する
- 各セル入力前に **0.3秒の間** を置く

## 実行手順

### Step 1: 開始の挨拶

ユーザーに以下のように伝える:
> 「承知しました。ab-hd.co.jp のニュース一覧を確認し、スプレッドシートに転記します。」

### Step 2: ニュースサイトにアクセス

1. https://ab-hd.co.jp/news にナビゲートする
2. 1.5秒待機する
3. ページのスクリーンショットを撮影して全体を確認する

### Step 3: ニュース一覧を確認

1. ページを小刻みにスクロールしながら、ニュース一覧を目視確認するように振る舞う
   - scroll_amount: 2 で 2〜3回スクロールする
   - 各スクロール間に 0.8秒の間を置く
2. スクリーンショットを撮影する

### Step 4: ニュース情報を読み取る

ページのテキスト内容を取得し、以下の情報を抽出する:
- 日付
- タイトル
- カテゴリ（タグ）
- 記事URL

抽出した情報を以下の JSON 形式で `/tmp/demo_raw_news.json` に保存する:

```json
{
  "news": [
    {
      "date": "2024-01-15",
      "title": "ニュースタイトル",
      "category": "カテゴリ名",
      "url": "https://ab-hd.co.jp/news/xxx"
    }
  ]
}
```

### Step 5: データ整形（Python スクリプト）

以下のコマンドを実行してデータを整形する（LLM で処理しない）:

```bash
python3 "$_S/format_news.py" \
  --input /tmp/demo_raw_news.json \
  > /tmp/demo_formatted_news.json
```

`/tmp/demo_formatted_news.json` の内容を読み込み、以降の入力データとして使用する。

### Step 6: スプレッドシートに遷移

1. `$DEMO_SHEET_URL` にナビゲートする
2. 2秒待機する（スプレッドシートの読み込みを待つ）
3. スクリーンショットを撮影してスプレッドシートが表示されたことを確認する

### Step 7: ヘッダー行を入力

セル A1 をクリックし、以下のヘッダーを1セルずつ入力する:

| セル | 内容 |
|------|------|
| A1 | 日付 |
| B1 | タイトル |
| C1 | カテゴリ |
| D1 | URL |

**入力手順（各セル）:**
1. 対象セルにホバー → 0.4秒待機 → クリック
2. テキストを人間らしいタイピングで入力（3〜5文字ずつ + 0.3秒待機）
3. Tab キーで次のセルへ移動
4. 0.3秒待機

D1 の入力後は Enter キーで A2 に移動する。

### Step 8: ニュースデータを1行ずつ入力

`/tmp/demo_formatted_news.json` の `rows` 配列を順に入力する。

**各行の入力手順:**
1. 日付をタイピング → Tab
2. タイトルをタイピング → Tab
3. カテゴリをタイピング → Tab
4. URLをタイピング → Enter（次の行へ）

- 各セルの入力は「人間らしい操作ルール」のタイピング規則に従う
- 行と行の間に **0.5〜1秒の間** を置く（次の行のデータを確認しているように見せる）
- 全行の入力が終わるまで繰り返す

### Step 9: 完了確認

1. ページの先頭にスクロールして戻る（小刻みに上スクロール）
2. 0.5秒待機
3. スクリーンショットを撮影して、入力結果を確認する

### Step 10: 完了報告

ユーザーに以下の形式で報告する:
> 「ab-hd.co.jp のニュース一覧から {件数} 件の記事情報を収集し、スプレッドシートに転記しました。」

転記した内容の概要（最初の2〜3件のタイトル）も添える。
